Services Offered by Firebase


Figure from miro.medium

Key Features

  • Cross-platform Compatibility: Android, iOS, Web, Cloud, IoT
  • Authentication: Flexible (Email/Password, Social Networks) or Standardized (Oauth 2.0, OpenID Connect)
  • Database: Cloud-based, NoSQL, Offline mode, Syncs real-time with clients, Stored in JSON
  • Storage: Supports many file formats, Compatible with Google Cloud Storage (scalable, shareable)
  • Hosting: Static web content, CLI tools, Free SSL built in, CDN, Versioning and roll-backs, Hosted at *.firebaseapp.com
  • Remote Config: Test lab, Crash reports, Analytics
  • Notifications: Uses Firebase Cloud Messaging (FCM) to send notifications for user and data for app


Figure from stackshare





Limitations

  • Privacy: You don't own your own data, impossible to export your user data
  • Languages: Limited languages so you can't create your own custom back-end API
  • Pricing: Only paid version, you can't set up own server, expensive fees for exceeding monthly limits
  • Security: Complex security protocols











Services Offered by Realm


Figure from Realm



Key Features

  • Two SDK's: Realm Mobile Platform (cloud) & Realm Mobile Database (on-device)
  • Built from ground up to run on mobile devices
  • Offline first
  • True live data objects
  • Seamless synchronization using sockets
  • Alternative to SQLite and Core Data
  • Supports a wide variety of languages, including Java, Swift, C, JavaScript, and .NET







Figure from Realm




Comparison to Other Databases

  • Like Core Data or SQLite, but without steep learning curve
  • Compatibility with wide range of languages makes it easy to learn
  • Like other databases, data can be queried, filterd, and interconnected
  • Unlike traditional databases, builds as true objects database, treating data objects as actual live objects
  • Unlike other platforms, is an offline first service, which enables users to access on-device database with or without network connection
  • Seamless synchronization using sockets, even after offline mode








Services Offered by Microsoft Azure


Figure from Microsoft





Key Features

  • Uses Microsoft's own tools and services (where other companies might outsource their cloud services)
  • Claims their price is five times lower than AWS, and offers price match
  • Supports authentication, push notification services, and database
  • Provides highly scalable development platform
  • Directly provides corresponding API's to database queries
